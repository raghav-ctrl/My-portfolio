## ℹ️ About

This is the personal portfolio of **T. Raghvendra Rahul**. Built with Next.js, Tailwind CSS, and GSAP.

## 🛠 Installation & Set Up

First, run the development server:

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
